Làm quen với ES6: Các tính năng hiện đại của JavaScript

JavaScript là một trong những ngôn ngữ lập trình phổ biến nhất trên thế giới. ES6 (ECMAScript 2015) đã ra đời như một bước ngoặt lớn, mang đến một loạt cải tiến và công cụ mới cho các nhà phát triển.

Tại sao ES6 là bước ngoặt lớn?

Trước ES6, việc viết mã JavaScript phức tạp thường đòi hỏi nhiều workaround và thư viện bổ trợ. ES6 đã thay đổi điều này bằng cách bổ sung những tính năng hiện đại, giúp mã ngắn gọn, dễ hiểu, và hiệu quả hơn.

Các tính năng nổi bật của ES6

1. Let và const

Trước ES6, JavaScript chỉ có từ khóa var để khai báo biến, dẫn đến nhiều vấn đề về phạm vi (scope) và khả năng dễ bị lỗi. ES6 giới thiệu let và const:

  • let: Khai báo biến có phạm vi block, có thể thay đổi giá trị.
  • const: Khai báo biến có phạm vi block, không thể thay đổi giá trị.
    • if (true) {
            let x = 10;
            const y = 20;
            console.log(x, y); // Output: 10, 20
        }
        // console.log(x, y); // Lỗi: x và y không tồn tại ngoài block

    2. Arrow Functions

    Arrow functions cung cấp cách viết hàm ngắn gọn hơn và tự động bind từ khóa this.

    // Hàm truyền thống
        function add(a, b) {
            return a + b;
        }
        
        // Arrow function
        const add = (a, b) => a + b;
        
        console.log(add(5, 10)); // Output: 15

    3. Template Literals

    Template literals cho phép tạo chuỗi dễ dàng hơn với cú pháp sử dụng dấu backtick (). Bạn có thể nhúng biểu thức vào chuỗi bằng ${}. 

    const name = "John";
        const age = 30;
        
        const message = `Hello, my name is ${name} and I am ${age} years old.`;
        console.log(message);

    4. Destructuring

    Destructuring giúp tách giá trị từ mảng hoặc đối tượng một cách dễ dàng.

    // Với mảng
        const [a, b] = [1, 2];
        console.log(a, b); // Output: 1, 2
        
        // Với đối tượng
        const person = { name: "Alice", age: 25 };
        const { name, age } = person;
        console.log(name, age); // Output: Alice, 25

    5. Modules

    ES6 cung cấp cú pháp native để export và import module.

    // file math.js
        export const add = (a, b) => a + b;
        export const subtract = (a, b) => a - b;
        
        // file main.js
        import { add, subtract } from './math.js';
        
        console.log(add(10, 5)); // Output: 15
        console.log(subtract(10, 5)); // Output: 5

    Kết luận

    ES6 đã thay đổi cách viết mã JavaScript, giúp nó trở nên hiện đại và mạnh mẽ hơn. Các tính năng như let, const, arrow functions, template literals, destructuring, và modules không chỉ làm cho mã ngắn gọn mà còn dễ bảo trì hơn.
